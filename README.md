Patricks Dotfiles
===================

These are my dotfiles to configure my development environment.

## Installation

Clone this repo to your **home** directory (`/Users/username`).
```
git clone https://github.com/Patro/dotfiles ~/.dotfiles
cd ~/.dotfiles
./install
```
